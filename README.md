# Quantium Data Analytics Virtual Experience Program

## Overview

This repository contains my work completed as part of the **Quantium Data Analytics Virtual Experience Program** on Forage. The project focuses on customer analytics, retail transaction analysis, experimentation, and business reporting using real-world retail datasets.

## Objectives

* Analyze customer purchasing behaviour.
* Identify key customer segments and product preferences.
* Evaluate the performance of trial stores against control stores.
* Generate data-driven business recommendations.
* Present insights in a professional client report.

## Task 1: Customer Analytics & Insights

### Key Activities

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Customer segmentation analysis
* Brand performance analysis
* Pack size analysis
* Data visualization

### Tools Used

* Python
* Pandas
* NumPy
* Matplotlib

### Key Findings

* Kettle was the highest-performing chip brand.
* Family-oriented customer segments contributed significantly to sales.
* Mainstream customers generated the largest share of revenue.
* Larger pack sizes were popular among family households.

---

## Task 2: Experimentation and Uplift Testing

### Key Activities

* Monthly sales analysis
* Trial vs Control store comparison
* Correlation-based control store selection
* Performance evaluation of trial stores
* Sales uplift assessment

### Trial Stores

* Store 77
* Store 86
* Store 88

### Analysis Metrics

* Total Sales Revenue
* Number of Customers
* Average Transactions per Customer

### Outcome

The analysis assessed whether trial stores performed better than comparable control stores and identified the drivers behind any observed sales uplift.

---

## Task 3: Client Reporting & Business Recommendations

### Deliverables

* Executive summary
* Data visualizations
* Key insights and findings
* Strategic recommendations

### Recommendations

* Expand successful trial initiatives to similar stores.
* Focus promotions on high-performing brands.
* Target family-based customer segments.
* Continue monitoring customer purchasing behaviour.
